To run the program:
+++Installing POD
	1.install cocoapods:
	sudo gem install cocoapods  

	2.initialize pod:(create Podfile with default)
	pod init  
	3.Copy these into Podfile:
  		pod 'FloatingPanel'
  		pod 'Firebase/Core'
  		pod 'FirebaseMessaging'
  		pod 'Firebase/Auth'
  		pod 'Firebase/Storage'
  		pod 'Firebase/Database'
		pod 'FacebookCore'
  		pod 'FacebookLogin'
  		pod 'Alamofire', '~> 4.4'
  		pod 'AlamofireImage', '~> 3.3'
  		pod 'SwiftKeychainWrapper'
  		pod 'MessageKit'
	4.install pods:
		pod install
	5.run .xcworkspace file
